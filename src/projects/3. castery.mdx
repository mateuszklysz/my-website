## Castery

Nowoczesny klient do wspólnego oglądania filmów. Jest szybki, wygląda pięknie i jest open-source.
Planuje dodać synchronizacje wideo między dwoma użytkownikami, czat tekstowy oraz głosowy.

<br />
Uruchomienie serwera deweloperskiego

```shell
   npm start
```
