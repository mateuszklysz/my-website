Strona dzięki której sprawdzisz pogodę w swoim mieście.
Korzystam z darmowego API <a href="https://openweathermap.org/api" target="_blank">OpenWeatherMap</a> oraz <a href="https://mapbox.com" target="_blank">MapBox</a>.  
Pomimo tego, że strona jest dynamiczna, postanowiłem wykorzystać <a href="https://gatsbyjs.com" target="_blank">GatsbyJS</a>.

<p style={{ color: "#5472d3", "padding-left": "20px" }}> npm start</p>
